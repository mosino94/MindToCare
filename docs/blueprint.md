# **App Name**: GeoPresence

## Core Features:

- Real-time Active User Count: Display the total number of active users on the platform in real-time.
- User Authentication: Allow users to sign in securely using email.
- Geolocation Tool: AI-powered tool determines users country based on IP address using the 'iplocation' tool in the background; this location will not be used in any identifying way beyond displaying the appropriate country flag emoji next to a username.
- Flag and Name Display: Show each active user's name and country flag emoji next to their name after they sign in.
- Active Status Tracking: Track users' active status in real-time.

## Style Guidelines:

- Primary color: Indigo (#4B0082) for a sense of authority and global awareness.
- Background color: Light gray (#F0F0F0), almost white.
- Accent color: Teal (#008080) for interactive elements and highlights, to contrast the indigo without clashing.
- Body and headline font: 'PT Sans' for a balance of modern style with approachability.
- Use simple, clean flag icons to represent each country.
- Clean and straightforward layout that displays information intuitively.
- Subtle fade-in animations on content to improve user engagement without being intrusive.
